What is Web Technology?

Web technologies are tools and techniques used to create and maintain websites and web applications. The major technologies include:

  • HTML (HyperText Markup Language) - Used for creating and structuring web content.
  • CSS (Cascading Style Sheets) - Used for styling and designing web pages.
  • JavaScript - Used for adding interactivity and dynamic behavior to web pages.
  • PHP, Node.js, Python, and more for backend development.

Let's take a look at some examples and interactive programs for each technology.

HTML - Create a Simple Web Page

HTML is used to structure the content on the web. Here's an example of a simple web page created with HTML:

<!DOCTYPE html>
<html>
<head>
  <title>My First Web Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is a simple HTML page.</p>
</body>
</html>

CSS - Style a Web Page

CSS is used to style the layout and design of web pages. Here’s an example of how to change the color and font of text:

body {
  color: #333;
  font-family: Arial, sans-serif;
}

h1 {
  color: #4CAF50;
  text-align: center;
}

JavaScript - Add Interactivity

JavaScript is used to make web pages interactive. Below is an example of a simple function that adds two numbers:

function addNumbers(a, b) {
  return a + b;
}

var result = addNumbers(5, 10);
console.log(result); // Output: 15
